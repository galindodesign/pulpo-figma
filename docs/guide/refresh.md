# Updating your flow

## Change the diagram from the plugin form

Edit fields in the plugin, then click **Create flow** again.

Pulpo **replaces** the existing experiment diagram for that experiment — it removes the previous overview, journey, and variant frames before generating a new one. You do not need a separate refresh step.

Use **Create flow** when you:

- Add or change goals, touchpoints, or variants
- Update status, dates, or experiment details
- Fix something after your first pass

## Move frames on the canvas

If you drag Pulpo cards or touchpoints on the canvas, connector lines **update automatically** in the background. There is no **Refresh** button in the plugin today.

## Tips

- Start with just a title, then run **Create flow** again as you add detail.
- After moving nodes manually, zoom out and check that connectors still look right.

## Common mistakes

- Expecting a **Refresh** button — use **Create flow** to sync form changes to the canvas.
- Renaming the experiment to something completely new — Pulpo treats it as a new experiment and leaves the old frames on the canvas. Delete the old diagram manually if you no longer need it.
