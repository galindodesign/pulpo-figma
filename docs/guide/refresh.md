# Refresh vs create

Pulpo can **generate a new flow** or **update an existing one** — choose based on whether you already have a diagram on the canvas.

## Create flow

Generates the full experiment diagram from your current form data.

Use when:

- Building your first flow for an experiment
- You want to replace the existing Pulpo frames for this experiment

## Refresh

Updates connector lines and flow structure on an **existing** Pulpo diagram without starting from scratch.

Use when:

- You changed touchpoints, variants, or journey order
- Status or goals updated and the canvas should reflect it

The **Refresh** button appears when Pulpo detects an existing flow for the current experiment.

## Tips

- Prefer **Refresh** over creating duplicate flows on the same canvas.
- After major structural changes, review the canvas to confirm connectors look right.

## Common mistakes

- Clicking **Create flow** repeatedly and ending up with duplicate diagrams — use **Refresh** when updating.
