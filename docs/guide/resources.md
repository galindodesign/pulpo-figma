# Resources

Resources link your experiment diagram to **designs, tickets, and briefs** — so context stays in one place.

## Where to find it

**Experiment** section → Resources.

## How to use it

Paste full URLs (include `https://`). Pulpo detects the service and shows the right icon.

**Supported services:** Figma, Jira, Miro, Notion, Amplitude, Asana, Linear, GitHub, Slack, Trello, Confluence, Monday, ClickUp, and generic URLs.

## Expected result

Resource links appear in the **Resources** section of the experiment overview card on the canvas.

## Tips

- Link the Figma frame for each variant when you have separate designs.
- Add the experiment brief or Jira/Linear ticket so stakeholders can go deeper.

## Common mistakes

- Pasting partial URLs without `https://` — use the full link for reliable detection.
