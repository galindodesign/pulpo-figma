# Glossary

Terms used in Pulpo and growth experiment documentation.

| Term | Definition |
| --- | --- |
| **A/B test** | An experiment with two variants. Pulpo labels this automatically when a touchpoint has two variants. |
| **Guardrail metric** | A secondary metric that should not get worse while you optimize the primary metric (e.g. support contact rate). |
| **Journey** | The path users take from entry to exit, including touchpoints where variants may run. |
| **Multivariate test** | An experiment with three or more variants at the same touchpoint. |
| **Primary metric** | The metric that decides whether the experiment succeeded — always the **first goal** in Pulpo. |
| **Create flow** | Generates (or replaces) the experiment diagram from your current form data. |
| **Connector auto-update** | After you move Pulpo frames on the canvas, connector lines adjust in the background. |
| **Rolled-out variant** | The variant shipped to production after a concluded experiment — set via **Concluded** status and the **Rolled out** dropdown. |
| **Touchpoint** | A step in the user journey (e.g. landing page, checkout). Variants attach to touchpoints via **Add design variant**. |
| **Variant** | A treatment or design you're comparing at a journey touchpoint. |
