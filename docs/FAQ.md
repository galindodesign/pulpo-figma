# FAQ

## How do I install Pulpo?

In Figma: **Plugins → Manage plugins in Community** → search **Pulpo** → **Install**.

Or open [Pulpo on Figma Community](https://www.figma.com/community/search?resource_type=plugins&q=Pulpo).

## What's the minimum to create a flow?

Only the **experiment name**. See the [user guide](/USER_GUIDE) for what each field adds to the canvas.

## How do I mark a winner?

Set status to **Concluded**, choose the **rolled-out** variant, then mark the winner in the variants list.

## Can I have more than 2 variants?

Yes. Add as many variants as you need (A/B/C/… or multivariate).

## How do I update an existing flow?

Change fields in the plugin, then click **Create flow** again. Pulpo replaces the previous diagram for that experiment. See [Updating your flow](/guide/refresh).

## Which link types are supported?

Figma, Jira, Miro, Notion, Asana, Linear, GitHub, Slack, Trello, Confluence, Monday, ClickUp, and generic URLs.

## Why don't I see variant thumbnails?

Paste a **Figma frame link** on each variant (Copy link to selection on a frame). See [Variants](/guide/variants).

## Does Pulpo send my experiment data anywhere?

No. Pulpo runs inside Figma. Your experiment data stays in your Figma file — nothing is sent to external servers.

## Does Pulpo work in FigJam and Slides?

Yes — **Figma Design**, **FigJam**, **Figma Slides**, and **Buzz**.
