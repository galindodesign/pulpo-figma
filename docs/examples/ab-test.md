# A/B test example

## Context

Test whether a violet CTA button increases homepage conversion vs. the current neutral button.

## Inputs

**Experiment**
- Name: Homepage CTA: Button color test
- Status: Running
- Hypothesis: A higher-contrast CTA will increase click-through to signup

**Goals**
1. Primary: Conversion rate (CVR) — increase, target +5%
2. Guardrail: Bounce rate — decrease

**Journey**
- Entry → Landing page → **Pricing page** (touchpoint with variants) → Signup page → Exit

**Variants** (on the pricing page touchpoint)
- Blue button
- Red button

Add Figma frame links on each variant for thumbnails.

## Expected output

An experiment overview card, journey spine, and two variant cards branching from the touchpoint where you added them.
